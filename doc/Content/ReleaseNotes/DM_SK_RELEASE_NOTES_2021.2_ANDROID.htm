Release Notes for Dragon Medical SpeechKit 2021.2 (Android edition 3.10 R1)

New in version 2021.2 (3.10 R1)

  • Dragon Medical SpeechKit's minSdkVersion is 26. If your app uses a lower minSdkVersion, add the overrideLibrary attribute to the <uses-sdk> tag in your manifest. For more information, see: https://developer.android.com/studio/build/manifest-merge#override_uses-sdk_for_imported_libraries

Note: For information on previous releases, see: Release history.

Important information

  • Mobile operating system support: The current major Android version is supported along with the previous two major versions. This does not include beta or prevew releases. [HCPLAT-1485, DNB-32170]
  • Dragon Medical SpeechKit 2021.2 (Android edition) is compatible with server versions 3.5, 2020.x and 2021.x.
  • Dragon Medical Server 2021.2 is compatible with client versions 2020.x and 2021.x.
  • Filtering auto-texts (defining filters to deactivate auto-texts selectively) is deprecated and will no longer work.

Infrastructure

Infrastructure no longer supported

  • Dragon Medical SpeechKit and Dragon Medical Embedded 2021.2 applications do not work with version 3.5 servers.

  • Version 3.5 client applications do not work with version 2021.2 servers.

Infrastructure changes planned for the next release

  • Dragon Medical SpeechKit and Dragon Medical embedded 2021 applications will not work with version 3.5 servers.
  • Version 3.5 client applications will not work with version 2021 servers.

*End-of-life: third party environments

Support for third party environments is only valid as long as they are supported by the corresponding vendor and might be subject to other restrictions. Please contact Nuance Technical Support for details. For more information, see the documentation delivered with the third-party product and supporting Nuance documentation.

When standard support by the vendor has stopped, Nuance will continue support if an issue is specific to the Nuance solution, within the limitations of the vendor’s standard end-of-life and Nuance's policy. This means that issues that are a combination of the vendor’s component and the Nuance solution cannot be supported.

Dragon Medical SpeechKit product life cycle

Nuance provides support throughout the life cycle of one version and point release. Please note that Nuance will not perform problem analysis after the scheduled start of restricted maintenance. Dragon Medical Server versions released over three months after the dates stated here will not work with these client editions. For up-to-date information on these schedules, go to Nuance iSupport.

Release history

New in version 2020.4 (2020-12-03)

  • Android 11 is now supported.

  • Android 7 is no longer supported.

Solved in version 2020.4

  • Continuously dictating for a long time (> 5 minutes) no longer causes the app to become unresponsive. [DNB-24018]

  • The microphone off voice command no longer causes the app to shut down unexpectedly. [DNB-28572]

New in version 3.5 release 1 (2019-09-19)

Operating system support:

  • Android 9 and 10 are now supported. [HCPLAT-776]
  • Android 6 and lower are no longer supported. [DNB-20393]

Note: Dragon Medical SpeechKit's minSdkVersion is 24. If your app uses a lower minSdkVersion, add the overrideLibrary attribute to the <uses-sdk> tag in your manifest. For more information, see: https://developer.android.com/studio/build/manifest-merge#override_uses-sdk_for_imported_libraries

Solved issues in version 3.5 release 1

  • Naming inconsistencies have been resolved. You will need to change the project name in your gradle file to :NuanceSpeechAnywhere as shown in the SDK sample apps. [DNB-17355]
  • The app no longer stops unexpectedly when the user taps the back button during dictation. [DNB-18482]

Start recording automatically

You can start recording automatically when a speech-enabled activity is displayed.

Prerequisites

You've speech-enabled an activity.

Procedure

  1. Override the onResume() callback method in your activity.
  2. In this callback method, call synchronize() on the VuiController instance.
  3. Call startRecording() on the session object.

Example

@Override

protected void onResume() {

super.onResume();

theVuiController.synchronize();

Session.getSharedSession().startRecording();

}

Remarks

  • Calling startRecording from other activity life cycle callback methods (such as onCreate(), onStart()) isn't supported.
  • The onResume() callback method is invoked when the activity is first loaded and also on other occasions; for example, when the user returns to the activity, or if the device is rotated and configuration changes aren't handled by your activity. Make sure to track the recording state so that you don't start recording unintentionally in onResume().

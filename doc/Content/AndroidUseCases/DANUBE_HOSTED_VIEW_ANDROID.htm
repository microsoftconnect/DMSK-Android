Personalization & help screen

The personalization & help screen is an HTML view that contains a variety of pages. To access it, the user taps the menu icon on the speech bar or says what can I say while recording. You can also make this view available to your users via your app.

When the view opens, the active form is still speech-enabled; however, the view is modal and a restricted set of voice commands is available.

Prerequisites

You've initialized a speech recognition session and speech-enabled your app.

Procedure

To make this view available to your users, call Session.getSharedSession().showForm(contentType) with contentType set to:

  • ContentType.Default – opens with the default page.
  • ContentType.VoiceCommands – opens with the What You Can Say page (listing available voice commands and auto-texts).
  • ContentType.EndUserHelp – opens with the Help page (the Dragon Medical SpeechKit end user help).
  • ContentType.AddWord – opens with the Manage Vocabulary page.
  • ContentType.AutoTexts – opens with the Manage Auto-texts page.

Remarks

  • For more information on including the Dragon Medical SpeechKit help and your own end user help, see: End user online help.
  • For more information on how the user opens the various pages via voice command and how to use them, see the Dragon Medical SpeechKit end user help.
